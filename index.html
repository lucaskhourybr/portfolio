Entregas Full - WebSite e APK

Aplicativo de Roteirização para Entregadores com API OpenCage O Aplicativo de Roteirização para Entregadores foi desenvolvido para otimizar as operações de entrega, especialmente para entregadores de plataformas como Mercado Livre e Shopee. Utilizando a API da OpenCage para geocodificação, o aplicativo permite que os entregadores insiram uma lista de endereços de entrega e o sistema calcule as rotas mais eficientes, considerando a distância e o tempo. A solução proporciona uma forma prática e eficaz de organizar as entregas, minimizando o tempo de deslocamento e melhorando a produtividade.

Projeto SCADA - Retrofit de Telas do Shopping de Porto Velho

Este projeto consiste no desenvolvimento de uma solução SCADA utilizando o software Elipse E3 para o retrofit das telas de controle no shopping de Porto Velho. O foco principal é a criação de novas telas de supervisão para os sistemas de fancoils, onde foram instalados variadores de frequência. O objetivo é otimizar o controle e monitoramento dos sistemas de climatização, garantindo maior eficiência energética e precisão nas operações. A implementação do projeto visa também melhorar a interface do usuário para facilitar a gestão e manutenção dos equipamentos..

Supervisório HomeAssistant

Este projeto visa o monitoramento contínuo de servidores e equipamentos como câmeras, computadores, gerenciadores e controladoras. O objetivo principal é identificar possíveis perdas de comunicação e falhas no sistema, registrando essas informações em um banco de dados para posterior análise e manutenção preventiva. Além disso, o projeto conta com uma integração API para o monitoramento de um sistema de reservatório, utilizando o Node-RED para gerenciar dados e eventos em tempo real. A solução proporciona um controle mais eficiente e automatizado, garantindo a operação contínua e a detecção precoce de problemas no ambiente de TI e nas infraestruturas monitoradas.

Inserção dos Endereços

Passo 1: Inserção do Endereço de Coleta Na tela inicial do aplicativo, o entregador deve inserir o endereço de coleta dos pacotes. Este endereço corresponde ao local onde os pacotes a serem entregues estão inicialmente armazenados, seja um centro de distribuição, um armazém ou um ponto de coleta.


Passo 2: Inserção dos Endereços de Entrega Após informar o endereço de coleta, o entregador adiciona os endereços de entrega dos pacotes. O aplicativo permite que múltiplos endereços sejam inseridos, facilitando a gestão de várias entregas em uma única viagem.

×

Calcular Rotas

Passo 3: Com os endereços de coleta e entrega inseridos, o aplicativo utiliza algoritmos de otimização de rotas para calcular a sequência mais eficiente para realizar as entregas, considerando fatores como distância e tempo.


Passo 4: A rota otimizada é exibida no aplicativo, e o entregador pode seguir a sequência de entregas para realizar as entregas de forma rápida e eficiente.


Bônus: O entregador pode selecionar rotas já realizadas, útil quando é necessário retornar para uma nova tentativa de entrega, bastando marcar o checkbox. Além disso, ao clicar no ícone de mapa ao lado de cada rota, o entregador é redirecionado automaticamente para o Google Maps com o endereço já preenchido, facilitando a navegação.
